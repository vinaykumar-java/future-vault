# Future Vault 🔐

A secure Spring Boot application that allows users to write messages to their future selves. Notes remain locked until the selected unlock date and are automatically delivered via email.

## 🚀 Features

- User Registration
- User Login
- JWT Authentication
- Spring Security
- BCrypt Password Encryption
- Create Future Notes
- Update Notes
- Delete Notes
- View Personal Notes
- Email Scheduler
- Global Exception Handling
- Swagger API Documentation
- MySQL Database

## ▶️ Run the Project

Clone the repository:

```bash
git clone https://github.com/vinaykumar-java/future-vault.git
```

Go to the project folder:

```bash
cd future-vault
```

Run the project:

```bash
mvn spring-boot:run
```
